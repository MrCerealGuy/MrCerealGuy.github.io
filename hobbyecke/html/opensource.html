Elektrotechnik/Elektronik

LTspice
LTspice ist ein professioneller SPICE Simulator des bekannten Halbleiterherstellers Linear Technology.

Logisim
Logisim ist ein kostenloses Programm zur einfachen digitalen Schaltkreissimulation.

Open Source Engineering Tools
Software, die das Leben eines Ingenieurs einfacher machen.

KiCAD
KiCAD ist eine CAD-Software zur Erstellung von Leiterplatten.

gEDA
Das gEDA-Prjekt stellt eine Sammlung freier und quelloffener Software für die Entwicklung von elektronischen Schaltungen bereit.

 

 

Physik

OSP - Open Source Physics
Sammlung von Programmen, die physikalische Vorgänge anschaulicher machen.

 

 

Mathematik, Modellierung und Simulation

Sage: Open Source Mathematics Software
Sage ist ein freies Open Source Mathematiksystem das unter der GPL steht. Sage vereint die Funktionalität diverser Open Source Pakete unter einer gemeinsamen auf Python basierenden Oberflaeche.

Gnuplot
Gnuplot ist ein skript- bzw. kommandozeilengesteuertes Computerprogramm zur grafischen Darstellung von Messdaten und mathematischen Funktionen (Funktionenplotter).

GNU Octave
GNU Octave dient zur numerischen Lösung mathematischer Probleme, wie zum Beispiel Matrizenrechnung, das Lösen von (Differential-)Gleichungsystem, Integration und vieles mehr.

Scilab
Scilab ist ein umfangreiches Software-Paket für Anwendungen aus der numerischen Mathematik, der Funktionsumfang kann zu großen Teilen mit dem von MATLAB/Simulink verglichen werden.

Maxima
Maxima ist ein Computeralgebrasystem (CAS).

SciPy.org
SciPy ist eine Open-Source Software für Mathematik, Wissenschaften allgemein und Ingenieurwissenschaften.

MayaVi Data Visualizer
Eine Software zur Visualisierung von wissenschaftlichen Daten.

VisIt
Eine Software zur Visualisierung von wissenschaftlichen Daten.

SCIENTIFIC COMPUTING AND VISUALIZATION
Eine Linksammlung zum Thema wissenschaftliches Rechnen und Visualisierung.

MathJax
MathJax ist eine JavaScript-Bibliothek zum Anzeigen mathematischer Formeln in allen modernen Browsern.

Qfsm
Ein grafisches Tool für die Modellierung und Simulation von Zustandsautomaten.
